About the course
Курс «Основы HTML и CSS» — это первый шаг к профессии веб-разработчика.

HTML и CSS — языки, с помощью которых верстают страницы в интернете.

Благодаря интерактивным практическим заданиям курса овладеть основами 
этих языков совсем нетрудно.

Курс является вспомогательным для очных лекций проекта «Mindcraft» 
(www.mindcraft.pro) и будет пополняться новыми уроками и задачами.
