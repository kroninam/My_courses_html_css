About the course
На этом курсе ты узнаешь как работает интернет, 
научишься создавать сайты с помощью HTML и CSS 
и размещать их в сети. Попутно освоишь основные 
инструменты веб-разработчика: редактор кода, 
отладчик, Google.

Не бойся страшных непонятных слов! Курс создан 
специально для людей, которые никогда не программировали 
и ничего не знают о работе интернета.

Этот курс - инкарнация классического оффлайнового 
курса от ITC, который проводится каждый год в РЭУ 
им. Плеханова для всех желающих студентов бесплатно. 
Мы решили разместить материалы курса на Степике. 
Он содержит все материалы и задания, которые 
мы даём нашим студентам.
